Java Web目录结构

WEB-INF/web.xml：核心配置文件，映射Servlet类路径。
WEB-INF/classes/：存放编译后的Java类文件。
利用路径穿越（../../）可读取敏感文件。
XXE漏洞原理

XML解析器未禁用外部实体时，可通过file://协议读取本地文件，或通过HTTP请求外带数据（Blind XXE）。
Apache POI 3.10版本默认未禁用外部实体，导致漏洞。
Excel文件结构

XLSX为ZIP压缩包，内含XML文件，修改[Content_Types].xml可注入恶意实体。
漏洞利用链设计

文件上传 → 触发XXE → 外带数据 → 获取Flag。
