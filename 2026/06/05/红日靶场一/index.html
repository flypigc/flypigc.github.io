net session
net session 命令的核心作用是展示 “会话连接”，具体能查到以下关键信息：

  • 远程计算机名称 / IP：显示哪个设备（比如另一台电脑、服务器）正在连接你的电脑。
  • 连接类型：通常是 “Microsoft Windows Network”，即通过 Windows 网络共享建立的连接。
  • 连接状态：显示连接是 “已建立” 还是 “断开” 等状态。
  • 登录用户名：显示远程设备用哪个账号登录到你的电脑（仅部分场景显示）。
